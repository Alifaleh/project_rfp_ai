## Odoo Module Scaffold:

A scaffold to generate new odoo modules.

### Installation Guid

To install this module, Follow the following steps:

 - Add the module to Odoo addons directory.
 - Restart Odoo server.
 - Go to odoo Apps. Enter debug mode and click on update app list.
 - Search with the app name and click install to install the module.

For more info about the module, please don't hesitate to contact us on our official email [info@Scaffold.live](mailto:info@Scaffold.live?subject=Help%20With%20Odoo%20Modules).
